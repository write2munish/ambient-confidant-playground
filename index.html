Troubleshooting & Configuration Guide

        

          

            
1. Enable Local CORS on macOS

            
If you encounter connection issues, close Ollama and restart it in terminal enabling origins:

            
OLLAMA_ORIGINS="*" ollama serve

          

          

            
2. Enable Local CORS on Windows

            
Close Ollama from system tray, open PowerShell, and start Ollama with CORS origins enabled:

            
$env:OLLAMA_ORIGINS="*"
ollama serve

          

          

            
3. Running the Local Server

            
Launch the server by double-clicking run.sh (Mac) or run.bat (Windows) inside the directory, or run:

            
python3 -m http.server 8000
